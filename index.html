What is this?

The 3RWW Rainfall Explorer Prototype is a live preview of a prototype for the next version of 3 Rivers Wet Weather's Calibrated Radar Rainfall tool. It utilizes the new 3RWW Rainfall API.

This project is under active development. Source code is available on GitHub.
